**MyNotes - JavaScript Notes App**

A simple and interactive notes application built with HTML, CSS, and vanilla JavaScript. Users can create, view, edit, delete, favorite, and search notes, with data saved in the browser's localStorage for persistence.

This project demonstrates my understanding of DOM manipulation, state management, event delegation, and localStorage persistence.

**Demo**
You can see a live demo of this app here:
https://amooayomikun.github.io/JavaScript-Note-App/

**Features**

**1.Create Notes:** Add new notes with title and content.

**2.Edit Notes:** Edit existing notes without creating duplicates.

**3.Delete Notes:** Remove unwanted notes.

**4.View Notes:** Click on a note to view it in a modal (read-only mode).

**5.Favorite Notes:** Mark notes as favorites with a heart icon toggle.

**6.Filter Favorites:** Show only favorite notes.

**7.Search Notes:** Search through all notes by title or content.

**8.Persistent Storage:** Notes are saved in localStorage, so they remain after page reload.

**9.Dynamic UI Updates:** All actions update the interface instantly without page reload.

**Usage**

1  Click “+ Add New Note” to create a new note.

2  Fill in the title and content. Click Save.

3  Click a note card to view it in read-only mode.

4  Click the Edit button to modify a note. Changes are automatically saved when you finish editing.

5  Click the heart icon to mark/unmark a note as favorite.

6  Click Favorites in the sidebar to see only favorite notes.

7  Use the search bar to filter notes by title or content.

8  Click the Delete button to remove a note permanently.
