**MyNotes - JavaScript Notes App**

A simple and interactive notes application built with HTML, CSS, and vanilla JavaScript. Users can create, view, edit, delete, favorite, and search notes, with data saved in the browser's localStorage for persistence.

This project demonstrates my understanding of DOM manipulation, state management, event delegation, and localStorage persistence.
